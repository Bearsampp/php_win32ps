# php_win32ps

Win32ps is a tiny module which utilizes PSAPI (Process Status Helper) to get information about global memory usage and process specific memory and cpu time utilization.

**PHP 8.3 ✅ | PHP 8.4 ✅**

This is a fork of [crispy-computing-machine/php_win32ps](https://github.com/crispy-computing-machine/php_win32ps) maintained to keep things more up-to-date.
